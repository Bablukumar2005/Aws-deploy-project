# AWS S3 Static Website Hosting

## Project Title
Deploying a Live Website on AWS Cloud Using Amazon S3

---
Link:-http://bablukr-website-2026.s3-website.ap-south-1.amazonaws.com

## Objective
The objective of this project is to deploy a static website on AWS Cloud using Amazon S3 Static Website Hosting.

---

## Technologies Used
- HTML
- CSS
- JavaScript
- AWS S3

---

## Files Used
- index.html
- style.css
- script.js

---

## Steps Performed

1. Created an AWS account.
2. Opened Amazon S3 service.
3. Created a new S3 bucket.
4. Disabled Block Public Access.
5. Uploaded website files.
6. Enabled Static Website Hosting.
7. Added Bucket Policy for public access.
8. Accessed the website using the live endpoint URL.

---

## AWS Services Used
- Amazon S3

---

## Website Features
- Responsive webpage
- Simple UI design
- Hosted on AWS Cloud
- Publicly accessible using live URL

---

## Output
The website was successfully deployed on AWS S3 and is accessible through a live public URL.

---

## Conclusion
This project helped in understanding cloud computing concepts, AWS S3 bucket management, static website hosting, and public deployment of websites on AWS Cloud.
